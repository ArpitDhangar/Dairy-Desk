# 🥛 Dairy Desk — Dairy Management System

🚀 A full-stack web application designed to **digitize and simplify dairy business operations**, including customer tracking, daily milk records, and automated monthly billing.

🔗 **Live Demo:** https://dairy-desk-inky.vercel.app/

---

## 📌 Problem Statement

Traditional dairy businesses often rely on **manual record-keeping**, which leads to:

* Errors in calculations
* Difficulty in tracking customer transactions
* Time-consuming monthly billing

---

## 💡 Solution

**Dairy Desk** solves these problems by providing a **digital platform** to:

* Track daily milk/product entries
* Manage customer accounts
* Automatically calculate monthly bills
* Reduce manual effort and errors

---

## ✨ Features

* 👤 Customer Management
* 🥛 Daily Milk Entry Tracking
* 📊 Monthly Billing System

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/dairy-desk.git
cd dairy-desk
```

### 2️⃣ Setup Backend

```bash
cd backend
npm install
npm run dev
```

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🌍 Environment Variables

Create a `.env` file in frontend:

```env
VITE_API_URL=http://localhost:5000/api
```

And backend:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
```
